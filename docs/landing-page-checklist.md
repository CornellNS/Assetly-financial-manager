# Assetly Download Landing Page Checklist

## Requirements

- [x] Product name is the first-viewport headline: Assetly Financial Manager.
- [x] Page is structured to market and sell downloadable desktop software.
- [x] Primary CTA downloads the app instead of opening the workspace.
- [x] Hero uses real Assetly product imagery, not generic stock finance artwork.
- [x] Download section clearly shows available and coming-soon platforms.
- [x] macOS Apple Silicon ZIP is stored in private Blob storage and served through checkout-confirmed, expiring download links.
- [x] Buyers can recover download access by email after losing the original browser session.
- [x] Windows installer card matches the current release state and routes to checkout with the Windows platform selected.
- [x] Copy explains the core promise: private, local-first personal finance management.
- [x] Page keeps the current Assetly design language: paper surfaces, thin borders, sage/amber/blue tones, compact cards, tabular numbers, 8px radii.
- [x] Motion is provided by `motion` and respects reduced-motion preferences.
- [x] Page is responsive for mobile, tablet, laptop, and wide desktop.
- [x] Interactive controls have accessible labels, focus rings, and touch-sized targets.
- [x] Images include dimensions/alt text through Next image optimization.
- [x] No unsupported financial, investment, tax, or security guarantees are made.
- [x] Desktop Electron app still opens directly into the workspace instead of the marketing page.

## Feature Coverage

- [x] Net worth and monthly finance overview.
- [x] Cash flow and weekly report positioning.
- [x] Debt and credit-card payoff tracking.
- [x] Stocks, crypto, lots, gains, and market snapshots.
- [x] Tax planner and yearly report workflows.
- [x] Local browser storage and recovery posture.
- [x] AI assistant framed as review-and-approve support, not automatic money movement.
- [x] Desktop packaging/release readiness mentioned without pretending a public download exists yet.
- [x] Comparison section explains why Assetly is better than spreadsheets or basic bank apps.
- [x] FAQ handles download, platform, privacy, and advice disclaimers.

## UX Checks

- [x] One primary action per major section: download.
- [x] Feature cards are scannable and do not nest cards inside cards.
- [x] Product screenshots remain visible and inspectable.
- [x] Landing page has a single-page anchor structure.
- [x] Footer includes download, FAQ, and a secondary demo workspace link.
